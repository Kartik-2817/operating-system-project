# operating-systen-project

OS Algorithm Visualizer is an interactive web-based tool built using HTML, CSS, and JavaScript that helps users understand the working of key Operating System algorithms through real-time visualizations and animations.

It allows users to input custom data, visualize step-by-step execution, and analyze algorithm performance using Gantt charts and dynamic animations.

🔍 Features

🎯 CPU Scheduling Algorithms (FCFS, SJF, Round Robin, Priority, etc.)

💾 Disk Scheduling Algorithms (FCFS, SCAN, C-SCAN, LOOK, C-LOOK)

🧠 Memory Management Algorithms (Paging, Segmentation, Page Replacement)

📊 Interactive Gantt Chart Visualization for process execution timelines

⚙️ User Input Support – customize processes, burst times, and priorities

🎨 Smooth Animations with a clean and responsive UI

💡 Educational Focus – perfect for students to learn OS concepts visually

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Libraries: Vanilla JS (no external dependencies)

Visualization: Custom animations + dynamic DOM rendering

🚀 Purpose

This project is designed to make complex OS algorithms easy to understand by combining visual learning with hands-on interaction, helping students and developers grasp core scheduling and memory management concepts effectively.
